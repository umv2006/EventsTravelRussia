# EventsTravelRussia
Описание формата для загрузки событий в Календарь событий.
# Мероприятия делового характера
К событиям делового характера относятся все мепроприятия, которые ориентированы на участие представителей туристического рынка.

Для мероприятий, направленных не только на туристический рынок, обязательным для включения календаря является проведение отдельной секции по туризму
## Пример описания события 
Событие является вымышленным и приведено для примера
```
Название: Мастера гостеприимства
Дата проведения события: 13.04.23-15.04.23,17.04.23
Отображение: public
Формат события: offline
Точность даты: approximate
Комментарий к дате: Планируется к проведению в марте
Регион проведения: Пермский край
Масштаб события: national
Место проведения: Пермь
Тип события: Конкурс
Дополнительная информация: https://vk.com/welcomecupru,https://welcomecup.rsv.ru
Целевая аудитория: Специалисты индустрии гостеприимства РФ
Стоимость посещения: Бесплатно
Организатор: ГБУ ПК «Центр развития туризма» info@visitperm.ru
Описание: Конкурс для талантливых и инициативных специалистов индустрии гостеприимства, направленный на создание благоприятных условий их развития и кадровое обеспечение туристской отрасли.
Проводится впервые.

-------------------
```

### Инструкция по заполнению данных
* Список событий вносится в текстовый файл (Блокнот, .txt расширение).
* Файл может содержать произвольное число событий. Каждое событие должно быть отделено от другого пустой строкой и несколькими знаками короткого тире
* Не допускается использование смайликов
* Не допустимо излишнее использование больших букв или других знаков препинания
* Адреса сайтов могут быть указаны только в разделе Сайты или Контактов организаторов
* Контакты (электронная почта, телефоны и другое) должны размещаться только в разделе Организатор
* Почти все поля являются "опциональным", но чем больше вы внесете информации, тем более интересным окажется описание ваших событий, и тем более интересно оно будет для посетителей. Кроме того, не внесенные поля не попадают в "фильтры", что снижает число показов
 
| Поле | Требование | Как вносить |
| ------ | ---------- | ------- | 
| Код события | Опционально | не указывается для новых событий, которые еще не внесены в общий календарь. Для исправления данных, указывается код собыитя, тем самым оператор календаря понимает, что нужно исправить событие, а не вносить его как новое событие |
| Код основного события | Опционально | Если событие проходит в рамках другого события, уже внесенного в календарь, можно внести его код. События будут отображаться рядом |
| Название | Обязательно | Не более 70 знаков |
| Отображение | Опционально | Допускаются только следующие значения: **public** - ваше событие будет отображаться на многих сайтах или **private** - для отображения только под паролем для таких же, как вы профессионалов. Эта опция нужна, если вы планируете мероприятие, хотите избежать наложения с коллегами, но не можете по разным причинам его аннонсировать. По умолчанию - public
| Точность даты | Обязательно | Возможное значение: **exact** либо  **approximate** либо **estimated** (см ниже)|
| Даты | Обязательно | Даты перечисляются в полном формате, например, запись 02.12.2024 будет означать второе декабря 2024 года. Возможно перечисление дат через запятую, или периодами, когда вносится первая и последняя дата, например: 02.12.2024-05.12.2024,17.12.2024-18.12.2024,20.12.2024 |
| Формат события | Опционально | В поле возможно только следующие значения: **online** - для мероприятий, которые проводятся в интернете, **offline** - для очных мероприятий, и **mixed** - если мероприятие проводится как он лайн, так и оффлайн |
|Комментарий к дате| Опционально | Любой комментарий о датах проведения, не более 80 символов
|Масштаб события| Опционально | Допустимые значения: **international** или **national** или **multiRegional** или **regional** или **local** |
|Регион проведения| Обязательно | Допустимо только использование название региона из списка регионов, без сокращений. Если мероприятие проводится в том же регионе, где находится организация, можно не указывать (такое значение будет присвоено автоматически) 
|Место проведения| Опционально | Город, адрес и название места, где проводится мероприятие|
|Тип события| Опционально | Допустимые значения - из списка Типов событий. Только одно значение.
|Дополнительная информация| Опционально | Указываются через запятую сайты, где можно получить дополнительную информацию о мероприятии. Например: https://google.com,https://yandex.ru |
|Целевая аудитория| Опционально |Свободный текст до 80 знаков, кого вы ожидаете на мероприятии
|Стоимость посещения| Опционально | Допустимые значения только "бесплатно" или "платно"|
|Организатор| Опционально | Указывается организация, проводящее мероприятие
|Контакты организатора| Опционально | Указываются контакты организатора (телефон, сайт, электронная почта, другое)
|Медиа| Опционально | Указываются через запятую URL фотографий с вашего сайта, которые можно использовать для продвижения вашего события
|Описание| Опционально | Свободный текст до 500 знаков. Это единственное поле, которое может быть из нескольких параграфов
| | | **Обратите внимание, что блок события должен заканчиваться пустой строчкой и строчкой, содержащей несколько коротких тире**
|  |  |  | 

#### Точность даты
| Код | Расшифровка |
| ------ | ------- | 
|exact| **точно** дата точно определена. Это не печатный каталог, поэтому дату можно будет исправить в случае переноса мероприятия|
|approximate| **приблизительно**. Означает, что мероприятие точно состоится в указанные даты, либо промежуток времени. Допустимо указывать в датах 01.01.2024-31.12.2024 для мероприятий, планируемых для проведения в течение года |
|estimated| **предположительно**. Если само мероприятие только планируется к проведению, но даты не утверждены|

#### Масштаб события
| Код | Расшифровка |
| ------ | ------- | 
 | international | Международное |
 |national|Всероссийское|
|multiRegional|Окружное, то есть для ближайших регионов, либо Федерального округа|
|regional|Региональное|
local|Муниципальное, городское или местного значения|
